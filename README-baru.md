## Teknologi yang Digunakan

# SIFEDORA - Sistem Informasi Fasilitasi/Evaluasi Dokumen Perencanaan Kabupaten/Kota

SIFEDORA adalah platform digital berbasis web untuk mengelola proses fasilitasi dan evaluasi dokumen perencanaan (seperti RKPD, RPJMD, RPD) antara Pemerintah Provinsi Maluku Utara dengan Pemerintah Kabupaten/Kota.

## Fitur Utama

- **Role-Based Access Control (RBAC)**: Superadmin, Kaban, Admin PERAN, Tim Verifikasi, Tim Pokja, dan Kabupaten/Kota
- **Manajemen Master Data**: Jenis Dokumen, Kabupaten/Kota, Tahun Anggaran, Tim Pokja
- **Penjadwalan & Surat**: Jadwal Fasilitasi dan Surat Pemberitahuan
- **Workflow Permohonan**: Pengajuan, Verifikasi, Evaluasi, Approval, dan Monitoring
- **Upload Dokumen**: Upload dan checklist dokumen persyaratan
- **Dashboard Dinamis**: Berbeda-beda sesuai role user
- **Activity Logging**: Tracking aktivitas pengguna
- **Responsive Design**: Menggunakan template SNEAT Bootstrap 5

## Teknologi yang Digunakan

- **Backend**: Laravel 11
- **Frontend**: Blade Templates, Bootstrap 5, JavaScript
- **Database**: MySQL
- **Template UI**: SNEAT - Bootstrap 5 Admin Template
- **Package Tambahan**:
  - `spatie/laravel-permission` (Role & Permission Management)
  - `spatie/laravel-activitylog` (Activity Logging)
